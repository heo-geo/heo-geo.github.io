# 3D in Web

## Assets

### Models
- [pmndrs market](https://market.pmnd.rs/) — free 3D models for React Three Fiber projects
- [Kenney](https://www.kenney.nl/) — free game assets, models, and textures
- [Spase Models](https://spase.io/models) — 3D model library

### Textures & Materials
- [Poliigon](https://www.poliigon.com/) — textures, models and HDRIs for 3D rendering
- [3D Textures](https://3dtextures.me/) — free seamless PBR textures (diffuse, normal, displacement, occlusion, roughness)
- [Poly Haven](https://polyhaven.com/) — free HDRIs, textures and models
- [ambientCG](https://ambientcg.com/) — free textures, HDRIs and models
- [Matcaps library](https://github.com/nidorx/matcaps) — huge library of matcap PNG textures organized by color

### Fonts & Converters
- [Facetype.js](http://gero3.github.io/facetype.js/) — convert fonts to JSON for use in Three.js
- [HDRI to CubeMap](https://matheowis.github.io/HDRI-to-CubeMap/) — convert HDRI files to cubemap format
