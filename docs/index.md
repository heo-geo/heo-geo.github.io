# Hey there 👋

I'm **Heorhii (George) Terentiev**, and this is my 🌱 **digital garden** — a personal corner of the web where I plant ideas, collect resources, and let thoughts grow over time.

Unlike a blog, where everything is polished and chronological, this space is **alive**, a bit **messy**, and always **evolving** 

Here, you'll find:

- ✍️ Notes and ideas I'm exploring
- 📚 Useful resources I don't want to forget
- 🔧 Tools, frameworks, and code snippets
- 💭 Musings about tech, learning, creativity, and more

Some notes are fresh sprouts 🌱 — unpolished and raw. Others have matured 🍃 — expanded and refined.

## 🤔 Why a Digital Garden?

Because learning isn't linear.

Because thoughts deserve sunlight.

Because sharing is better than hoarding.

This isn't a place for perfection. It's a space for **connection** — between ideas, people, and curiosity.

## 📍 How to Navigate

There's no single path. Wander. Click. Backtrack. Revisit. Just like in a real garden.
